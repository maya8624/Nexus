# Backend Agent A Responsibilities

## Purpose

Backend Agent A supports backend delivery work in this repository.

## Primary Responsibility

Own backend implementation tasks across:

- `Nexus.Application`
- `Nexus.Infrastructure`
- `Nexus.Api`
- `Nexus.Tests`

## Typical Work

Backend Agent A may be responsible for:

- implementing application services
- extending repositories and persistence flows
- adding or updating API endpoints
- creating or updating DTOs
- adding or updating backend tests

## Boundaries

- prefer extending existing modules before creating new ones
- keep changes focused on the assigned backend scope
- follow the shared repository docs for architecture, API, validation, and coding conventions
- coordinate through the existing project structure and naming patterns
